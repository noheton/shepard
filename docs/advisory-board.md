## The board

Each persona is invoked on a defined stage transition (per `feedback_persona_audit_triggers.md`) — and increasingly as a **standing reasoning resource**: any non-trivial lean cites the relevant persona's lens. When a question warrants a durable record, the persona is dispatched as a full agent and writes its findings to `aidocs/agent-findings/persona-<slug>-<feature-id>-<round>.md`.

The board's vocabulary lives at `aidocs/00-doc-stages.md` — the unified lifecycle taxonomy that maps onto doc front-matter, backlog row status, and GitHub Issue labels byte-for-byte. One state machine, three surfaces.

## How the board operates

**Stage-transition audits** — when an aidocs doc flips from `feature-defined` to `audited-by-personas`, the relevant subset of personas fires per `feedback_persona_audit_triggers.md`. Default trio: Strategy Aligner, API Scrutinizer, RDM. Feature-specific personas layer on. Each writes durable findings at `aidocs/agent-findings/persona-<slug>-<feature-id>-<round>.md`.

**Standing reasoning resource** — for non-trivial leans formed mid-flight (not just at stage transitions), the AI consults the relevant persona's lens inside the reply, citing which lens informed the lean. Lightweight; no full agent dispatch.

**Findings anchor to elements** — every recommendation cites at least one element: a backlog row ID, a file path + line range, a commit hash, an aidocs §, an OpenAPI path, or a memory rule. Vibes-based findings are flagged for re-grounding.

**Failure-mode discipline** — when a persona's lens is missing from a non-trivial reasoning step, that's the bug. The board is the rigour behind every lean.

## Recent five-way audit

The most recent full-board moment was the **GH-PM adoption audit on 2026-05-23**: six questions about adopting the project's first formal GitHub-project-management policy. Five personas fired in parallel (Strategy, API Scrut, RDM, Reluctant Senior, Digital Native), each producing ~200-line findings docs with element-anchored votes. Result: 6/6 decisions resolved — 4/6 by 5-vote consensus, 2/6 by user choice between split persona votes. **Digital Native flipped one of Claude's leans** with concrete evidence (the `gh project create` 31-line demo). The synthesis lives at `aidocs/agent-findings/gh-pm-adoption-synthesis-2026-05-23.md`.
