# shepard-v2-client (Python, Kiota-generated)

Python client for shepard's `/v2/` API surface, generated by
[Microsoft Kiota](https://learn.microsoft.com/openapi/kiota/overview)
from the per-shelf OpenAPI document `/shepard/doc/openapi/v2.json`
(P4c, shipped).

This is the **CG1a** new-baseline client per
[ADR-0022](../../aidocs/63-architecture-decision-log.md#adr-0022). For the
upstream-compatible `/shepard/api/...` surface, use the OpenAPI-Generator
clients under `../../clients/` (CG1b).

## Install

```bash
pip install shepard-v2-client
```

(Published on PyPI by CG1c — until then, build locally from this folder.)

## Quick start

```python
import asyncio
from kiota_abstractions.authentication import AnonymousAuthenticationProvider
from kiota_http.httpx_request_adapter import HttpxRequestAdapter
from shepard_v2 import ShepardV2Client

async def main():
    auth = AnonymousAuthenticationProvider()           # or BearerToken / X-API-Key
    adapter = HttpxRequestAdapter(authentication_provider=auth)
    adapter.base_url = "https://shepard.example.com"
    client = ShepardV2Client(adapter)
    # fluent path-builder mirrors the URL:
    # /v2/dataobjects/{appId} -> client.v2.dataobjects.by_app_id(app_id).get()

asyncio.run(main())
```

## Regenerating

```bash
# Boot shepard locally so /shepard/doc/openapi/v2.json is reachable, then:
make -C .. generate-python
```

Pinned Kiota version: see `clients-v2/Makefile` (`KIOTA_VERSION`).
